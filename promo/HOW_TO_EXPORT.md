# How to export the promo tiles as PNG

## Option A — Chrome DevTools (recommended, free)

1. Open `promo-tile-440x280.html` in Chrome
2. Press F12 to open DevTools
3. Press Ctrl+Shift+P (Mac: Cmd+Shift+P) → type "screenshot" → select **Capture screenshot**
4. Save as `promo-440x280.png`
5. Repeat for `promo-tile-920x680.html` → save as `promo-920x680.png`

## Option B — DevTools device toolbar

1. Open the HTML file in Chrome
2. F12 → click the device toolbar icon (or Ctrl+Shift+M)
3. Set exact dimensions (440×280 or 920×680)
4. Capture screenshot

## Upload to Chrome Web Store
- Small promo tile: upload `promo-440x280.png` (required)
- Large promo tile: upload `promo-920x680.png` (optional, helps with featuring)
